HTML Tags and Descriptions

1. <h1> - <h6>
Heading tags that define headings with <h1> being the highest level and <h6> the lowest level.
2. <p>
Paragraph tag used for defining paragraphs of text.
3. <ul>
Unordered list tag used to create bulleted lists.
4. <ol>
Ordered list tag used to create numbered lists.
5. <img>
Image tag used to display images on a web page.
